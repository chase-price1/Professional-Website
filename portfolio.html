BTech Certification

My certification at BTech gave me a well rounded education in web and mobile development. I learned the basics of everything, but had a focus in front end development in my elective classes. Click the button below to view many examples of what I learned and what I am capable of.

In one of my classes we built a weather app that displays the current weather, current temperature, and high/low temperatures in Logan, UT by default. The location can be changed through the searchbar at the top of the screen. The app uses a free API tool to gather the information, and is built in HTML, CSS, and JavaScript.
While I was learning backend languages, we built a stock website that gives you recommendations on how to handle stocks. It pulls information from a free API service and then displays the recommendations in graphs. The assignment was built using HTML, CSS, PHP, MySQL, and Bootstrap.
